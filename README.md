# Modbus Address Range
 - Total 1 ~ 247 (0x01 ~ 0xF7)
 - [01 ~ 30 (0x01 ~ 0x1E)] PV Inverter
 - [31 ~ 60 (0x1F ~ 0x3C] PV Temperature
 - [61 (0x3D)] GHI
 - [71 ~ 80 (0x47 ~ 0x50)] POA
 - 
 - [90~100 (0x5A~0x64)] Structure Sensor
 - 
 - [200 (0xC8)] Wind Direction
 - [201 (0xC9)] Wind Speed
 - [202 (0xCA)] Ambient Temperature
